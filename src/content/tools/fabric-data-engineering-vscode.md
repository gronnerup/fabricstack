## What it does

The Fabric Data Engineering extension (the successor to the Azure Synapse VS Code extension, now Fabric-focused) lets you author and run Fabric notebooks and Spark Job Definitions from VS Code. You can execute and debug code on remote Spark pools, sync notebooks between local and remote, and explore lakehouse structure and table data.

## Why use it

Use it when you want a local, editor-based development loop for Fabric data engineering, writing Spark code with VS Code's tooling while running against Fabric compute.

## Getting started

Search "Fabric Data Engineering VS Code" in the Extensions view and install (`SynapseVSCode.synapse`).
