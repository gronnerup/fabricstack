## What it does

Tabular Editor 2 is a free, open-source desktop editor for Analysis Services Tabular and Power BI semantic models via the XMLA endpoint. It enables fast editing of measures, calculated columns, display folders, perspectives, and translations, plus C# scripting, the Best Practice Analyzer, and command-line automation for CI/CD.

## Why use it

Use it for rapid, keyboard-driven model editing and for scripting bulk changes and quality checks, including running BPA in build pipelines via its command line. It is a long-standing community staple for semantic-model development.

## Getting started

Download the signed executable from the GitHub releases page (`releases/latest`). It integrates with Power BI Desktop as an external tool.
