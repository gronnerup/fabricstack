## What it does

Measure Killer is a governance and optimization tool for Power BI and Microsoft Fabric. The free edition runs as a Power BI Desktop external tool that scans the open model and report to find unused measures, columns, and tables, showing the DAX before you delete anything. Paid editions add tenant-wide lineage, column-level usage tracking, access analysis, and best-practice violation detection across the Power BI service and Fabric.

## Why use it

Use it to safely trim bloated models and understand what is actually used downstream, reducing model size and maintenance surface and, in the paid tiers, mapping dependencies across an entire tenant.

## Getting started

Install the free version (MSI, portable, or Microsoft Store) and launch it from the External Tools ribbon in Power BI Desktop. Paid Enterprise and Automation editions connect to the Power BI service / Fabric tenant.
