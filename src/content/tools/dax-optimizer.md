## What it does

DAX Optimizer is a hosted web service that pinpoints performance bottlenecks in the DAX measures of Fabric, Power BI, and Analysis Services semantic models. It analyzes a model, surfaces issues with prioritized recommendations, tracks improvements over time, supports data obfuscation for privacy, and enables team collaboration. It can be used via the web app, a Tabular Editor 3 integration, or a command-line NuGet package.

## Why use it

Use it when you want systematic, repeatable DAX performance analysis with guidance on what to fix first, rather than manually profiling every measure. It is built by the team behind the SQLBI patterns and Tabular Editor.

## Getting started

Try the free Basic analysis at https://www.daxoptimizer.com/free/, or subscribe at daxoptimizer.com, then connect or upload a model to run an optimization analysis.
