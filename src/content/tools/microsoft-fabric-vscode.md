## What it does

The Microsoft Fabric extension is the official core VS Code integration for Fabric. You can sign in to a Fabric account, browse and manage workspaces, and create, open, rename, and edit Fabric items directly in the editor. It acts as an extensibility platform that other Fabric "satellite" extensions build on, supports Git-enabled workspaces and version control, opens Fabric SQL databases via the MSSQL extension, and integrates a Fabric MCP server for GitHub Copilot Chat.

## Why use it

Use it as the foundation for Fabric development in VS Code: the base layer that the data-engineering and user-data-functions extensions extend.

## Getting started

Search "Microsoft Fabric" in the VS Code Extensions view and install (`fabric.vscode-fabric`). Then press F1 and run **Fabric: Sign in**.
