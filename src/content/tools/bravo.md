## What it does

Bravo is a free, open-source companion tool for building and optimizing Power BI models. It analyzes model size and per-column memory, formats DAX (via DAX Formatter), creates and manages date tables with time-intelligence templates, exports data to Excel and CSV, and exports model metadata to VPAX files. It runs as a Windows desktop app and connects to Power BI Desktop as an external tool.

## Why use it

Use Bravo as an approachable, all-in-one helper for common modeling tasks: a lighter, guided alternative to assembling several specialized tools, well suited to analysts who want quick wins on model size and DAX formatting.

## Getting started

Download the Windows app from https://bravo.bi, then connect to an open Power BI Desktop model or a published dataset.
