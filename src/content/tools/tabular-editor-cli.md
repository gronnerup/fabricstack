## What it does

The Tabular Editor CLI (`te`) is a standalone, cross-platform command-line interface that brings Tabular Editor 3 capabilities to any terminal and automation context. It offers 50+ commands spanning the model lifecycle (editing, Best Practice Analyzer, VertiPaq analysis, querying, deploy, refresh, and test), plus an interactive guided mode for newcomers and a non-interactive mode for scripting and CI. It is distinct from the legacy `TabularEditor.exe` command-line flags.

## Why use it

Use it to run Tabular Editor workflows on Windows, macOS, or Linux build agents and developer machines without the desktop GUI.

## Getting started

Create a Tabular Editor account, download the CLI for your OS from https://tabulareditor.com/download-tabular-editor-cli, extract the `te` binary and add it to your PATH, then run `te --help`. It is free to evaluate during the Limited Public Preview (through 30 September 2026); a license will be required at general availability.
