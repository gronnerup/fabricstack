## What it does

Power BI Studio is a Visual Studio Code extension that manages a Power BI tenant through the Power BI REST API. It provides a workspace and artifact browser for operations such as takeover, refresh and refresh history, rebind, clone, and parameter changes; lets you modify datasets using TMDL; and supports running DAX queries and REST calls from notebooks. It also integrates with Power BI Desktop as an external tool.

## Why use it

Use it to perform tenant management and semantic-model edits from VS Code, keeping common Power BI operations and DAX exploration in your editor.

## Getting started

Search "Power BI Studio" in the VS Code Extensions view and install (`GerhardBrueckl.powerbi-vscode`).
