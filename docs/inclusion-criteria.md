# Inclusion criteria

fabricstack does not rank, rate, or recommend. Inclusion is curated against
these published criteria: tools that meet them are in; tools that don't, aren't.

## A tool must

1. Be specifically built for, or directly useful to, Microsoft Fabric and/or
   Power BI development.
2. Be publicly accessible: not internal-only, and not behind a private NDA or
   agreement.
3. Be a **tool**: a CLI, SDK, API, MCP server, library, framework, IDE
   extension, desktop app, web app, template, or hosted service. **Not** a blog
   post, course, book, video, tutorial, or other learning material.

## What is not required

- Documentation is not required.
- Active maintenance is not required; deprecated tools stay in the catalog,
  marked with a badge.
- A free tier is not required.

## Review

The site maintainer reviews each submission and either approves it (a pull
request is merged) or declines it (the issue is closed with a comment
referencing the criterion that wasn't met).

See [contributing.md](./contributing.md) for how to submit, and
[tag-conventions.md](./tag-conventions.md) for tagging guidance.
